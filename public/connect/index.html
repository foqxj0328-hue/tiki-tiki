방법

Minecraft Java Edition 1.21.1 환경에서 NeoForge와 아래 필수 모드를 설치하면 tiki tiki에 접속할 준비가 끝납니다.

01

NeoForge 설치

아래 NeoForge Installer를 내려받아 실행하고, Minecraft 1.21.1 클라이언트 설치를 완료합니다.

02

필수 모드 배치

나열된 모드 12개를 모두 내려받아 %appdata%\.minecraft\mods 폴더에 넣습니다.

03

NeoForge 프로필로 실행

Minecraft Launcher에서 설치된 NeoForge 1.21.1 프로필을 선택해 실행합니다. 서버 주소는 운영 준비 완료 후 이 페이지에 공개됩니다.

안전한 설치를 위해 파일명과 버전을 임의로 변경하지 마세요. 각 파일에는 검증용 SHA-256 해시를 함께 표시했습니다.
